# 發票產生器 (Invoice Generator) — 使用說明

## 系統需求

- **作業系統**：Windows 10 / Windows 11
- **必要軟體**：Microsoft Excel（需已安裝，用於匯出 PDF）
- **執行檔**：`InvoiceGenerator.exe`（不需安裝 Python）

---

## 快速開始

1. 下載並解壓縮 `InvoiceGenerator.zip`
2. 準備好匯入用的 Excel 檔案（格式說明見下方）
3. 雙擊執行 `InvoiceGenerator.exe`
4. 依照介面指示選擇檔案與輸出資料夾，按下「Generate Invoices」即可

---

## 匯入檔案格式說明

匯入檔案必須為 **Excel 格式（.xlsx）**，且包含以下兩個工作表（Sheet）：

---

### 工作表一：`invoice`（客戶主檔）

每一列代表一位客戶的基本資料與帳務資訊。

| 欄位名稱 | 說明 | 範例 |
|---|---|---|
| `客戶代碼` | 客戶唯一識別碼（必填） | HK0C142875 |
| `客戶姓名` | 聯絡人姓名 | 王小明 |
| `公司名稱` | 公司全名 | 台灣物流股份有限公司 |
| `客戶地址` | 公司地址 | 台北市信義區忠孝東路一段1號 |
| `客戶電話` | 聯絡電話 | 02-1234-5678 |
| `未稅價格` | 未含稅金額（新台幣） | 1500 |
| `開立日期` | 發票日期（可留空，留空則使用介面所選日期） | 2026/03/17 |

> 注意：第一列為欄位標題列，資料從第二列開始填寫。

---

### 工作表二：`bill_audit`（帳單明細）

每一列代表一筆貨運交易記錄，透過「客戶代碼」與工作表一關聯。

| 欄位名稱 | 說明 |
|---|---|
| `客戶代碼` | 對應 invoice 工作表的客戶代碼（必填） |
| `業務單號` | 貨件追蹤號碼 |
| `目的地` | 目的地國家 |
| `計費重` | 計費重量（kg） |
| `A價_掛號` | 掛號費用（TWD） |
| `A價_運費` | 運費（TWD） |
| `A價_附加費(TWD)` | 附加費（TWD） |
| `A價總額(TWD)` | 合計金額（TWD） |
| `到貨時間` | 到貨日期 |

> 同一個客戶代碼可以有多筆明細，程式會自動彙整。

---

### 匯入檔案準備步驟

1. 開啟 Excel，新增空白活頁簿
2. 將第一個工作表重新命名為 **`invoice`**
3. 在第一列填入欄位名稱（如上表），第二列起填入客戶資料
4. 新增第二個工作表，重新命名為 **`bill_audit`**
5. 在第一列填入欄位名稱（如上表），第二列起填入交易明細
6. 儲存為 `.xlsx` 格式

> 可參考附件 `import_sample.xlsx` 作為格式範本。

---

## 操作步驟

### 步驟一：選擇匯入檔案

點擊「Browse」按鈕，選擇已準備好的 `.xlsx` 匯入檔案。

### 步驟二：選擇輸出資料夾

點擊「Browse」按鈕，選擇要儲存產出發票的資料夾。

### 步驟三：選擇發票日期

在日期欄位設定發票開立日期（預設為當日）。
若匯入檔案的 `invoice` 工作表中已填入 `開立日期`，則以該日期為準。

### 步驟四：產生發票

點擊「**Generate Invoices**」按鈕，程式會開始處理。
處理過程中可在下方紀錄視窗查看進度。

### 步驟五：查看輸出

處理完成後，點擊「Open Output Folder」可直接開啟輸出資料夾。

---

## 輸出結果

每位客戶會產生以下兩個檔案：

| 檔案格式 | 命名規則 | 說明 |
|---|---|---|
| `.xlsx` | `{客戶代碼}_{日期}_invoice.xlsx` | Excel 發票（含服務費明細與交易記錄兩個工作表） |
| `.pdf` | `{客戶代碼}_{日期}_invoice.pdf` | PDF 版發票（供列印或傳送客戶用） |

範例：`HK0C142875_20260317_invoice.xlsx` / `HK0C142875_20260317_invoice.pdf`

## 常見問題

**Q：執行時出現「找不到 Excel」的錯誤？**
A：請確認電腦已安裝 Microsoft Excel，且版本支援 COM 自動化（Office 2013 以上建議）。

**Q：為什麼部分客戶沒有產出發票？**
A：請確認 `bill_audit` 工作表中，該客戶的「客戶代碼」與 `invoice` 工作表完全一致（區分大小寫）。

**Q：輸出的 PDF 版面跑掉？**
A：請確認 Excel 的頁面設定正確，並確保列印範圍未超出 A4 頁面。

**Q：程式執行完畢但找不到輸出檔案？**
A：請確認步驟二所選擇的輸出資料夾路徑，並確認該資料夾有寫入權限。

---

## 注意事項

- 執行期間請勿手動開啟或操作 Microsoft Excel，以免影響 PDF 匯出
- 匯入檔案中的欄位名稱必須**完全一致**，包含空格與大小寫
- 建議在匯入前先備份原始資料
